## 🎮 A MSX Emulator & Debugging Suite for Windows & Linux.

MSX+ is a modern emulator and development toolkit for the MSX 1,2 en 2+ systems, built with Qt6 and inspired by my ADAM+ project.
The software has been completely redesigned from the ground up, focusing on stability, performance, and expandability.

The MSX+ emulator is built using the latest available techniques and technologies obtainable in 2026. Leveraging deep expertise and the assistance of advanced
Language Models (LLMs), we can achieve the full potential of our programming skills with exceptional speed and accuracy.

For the moment only MSX 1 is supported, ROMS,DISKS & TAPES can be played.
